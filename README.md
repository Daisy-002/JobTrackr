# JobTrackr 🚀

A smart job application tracking dashboard built with React to help users manage, organize, and analyze their job search efficiently.

## 📌 Problem

Job seekers often apply to multiple roles across different platforms and struggle to track applications, interviews, and follow-ups. Traditional methods like spreadsheets lack visibility, automation, and analytics.

## 💡 Solution

JobTrackr provides a centralized dashboard to:

* Track all job applications
* Monitor interview progress
* Search, filter, and sort applications
* Visualize job search analytics

## ✨ Features

* Add, edit, and delete job applications
* Search with real-time filtering
* Sort by date, salary, or company
* Application pipeline (Applied, Interview, Offer, Rejected)
* Bookmark important jobs
* Dashboard with analytics and charts

## 🛠 Tech Stack

* React
* React Router DOM
* Axios
* React Hook Form + Yup
* Recharts
* Context API

## 🚀 Getting Started

```bash
npm install
npm run dev
```
